# Alignment / Calibration UX Requirements (User Feedback)

## Requested updates

1. **Keep Align visible at all times**
   - The **Align** button should remain visible even after successful alignment/calibration.

2. **Show explicit on-screen instructions**
   - The calibration flow should include clear, step-by-step instructions visible on screen.
   - The user should always know what to do next.

3. **Add alignment target toggle with icons**
   - Add a toggle control to choose alignment target.
   - Include a **sun icon** and **moon icon** options for target selection.
